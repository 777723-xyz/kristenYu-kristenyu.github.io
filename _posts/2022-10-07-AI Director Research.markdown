I spent some time working with Inflexion Games to implement an RL Based AI Director in their upcoming game Nightingale. Due to changes in the game development, the AI director is no longer implemented in the game. 
However, I did run a human subject study using Nightingale, to inconclusive results. As such, I developed FarmQuest, which more information can be found in my post about FarmQuest.
